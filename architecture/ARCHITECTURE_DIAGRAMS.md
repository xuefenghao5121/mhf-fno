## 二、CoDA 注意力机制详解

### 2.1 CoDA 内部结构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         CoDA 风格跨头注意力                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  输入: [B, n_heads=4, head_dim=8, H, W]                                    │
│                                                                             │
│  Step 1: 空间池化                                                           │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │   [B, 4, 8, H, W] ──mean(dim=(-2,-1))──► [B, 4, 8]                 │   │
│  │                                                                     │   │
│  │   每个头获取一个全局特征向量                                        │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  Step 2: 瓶颈压缩                                                           │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │   [B, 4, 8] ──Linear(8→4)──► [B, 4, 4]                             │   │
│  │                                                                     │   │
│  │   压缩率 50%，正则化效果，参数高效                                  │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  Step 3: 跨头注意力                                                         │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │   [B, 4, 4] 作为序列长度=4 的输入                                   │   │
│  │                                                                     │   │
│  │   MultiheadAttention(embed_dim=4, num_heads=2)                     │   │
│  │                                                                     │   │
│  │   ┌───────────────────────────────────────────────────────┐        │   │
│  │   │  Attention Matrix (4×4)                               │        │   │
│  │   │                                                       │        │   │
│  │   │         H1    H2    H3    H4                         │        │   │
│  │   │    H1 [0.4,  0.2,  0.2,  0.2]  ← H1 关注自己最多      │        │   │
│  │   │    H2 [0.1,  0.5,  0.2,  0.2]  ← H2 关注自己+邻居     │        │   │
│  │   │    H3 [0.1,  0.2,  0.5,  0.2]                         │        │   │
│  │   │    H4 [0.1,  0.1,  0.2,  0.6]  ← H4 关注自己最多      │        │   │
│  │   │                                                       │        │   │
│  │   │  学习频率头之间的交互模式                             │        │   │
│  │   └───────────────────────────────────────────────────────┘        │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  Step 4: 特征重建                                                           │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │   [B, 4, 4] ──Linear(4→8)──► [B, 4, 8]                             │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  Step 5: 门控融合                                                           │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │   delta = out_pooled - residual                                     │   │
│  │   out = x + gate * delta                                            │   │
│  │                                                                     │   │
│  │   gate 是可学习参数，初始值 0.1                                     │   │
│  │   控制注意力的增强强度                                              │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  输出: [B, 4, 8, H, W] (广播增量到空间维度)                                 │
│                                                                             │
│  参数统计:                                                                  │
│  ├─ 压缩层: 8×4 = 32                                                       │
│  ├─ 注意力: ~64 (MultiheadAttention 内部)                                  │
│  ├─ 重建层: 4×8 = 32                                                       │
│  ├─ 门控: 1                                                                │
│  └─ 总计: ~130 参数 (<0.1% 模型)                                           │
│                                                                             │

## 三、频率耦合原理图

### 3.1 NS 方程的频率特性

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    Navier-Stokes 方程频率耦合                                │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  方程: ∂u/∂t + (u·∇)u = ν∇²u                                               │
│              ↑              ↑                                               │
│         对流项(非线性)    扩散项(线性)                                      │
│                                                                             │
│  频率空间分析:                                                              │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │   低频 (大涡) ◄────────────────────────────────────► 高频 (小涡)   │   │
│  │       │                    能量级联                     │           │   │
│  │       │                  (Kolmogorov)                   │           │   │
│  │       │                     ↓                          │           │   │
│  │       └──────────────► 频率混合 ◄───────────────────────┘           │   │
│  │                          ↑                                          │   │
│  │                    非线性对流项                                      │   │
│  │                                                                     │   │
│  │   数学表示:                                                         │   │
│  │   FFT{(u·∇)u} = ∫ û(k') û(k-k') dk'  ← 卷积 = 频率混合             │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  Darcy Flow 对比:                                                           │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │   -∇·(a∇u) = f  (椭圆型)                                           │   │
│  │                                                                     │   │
│  │   频率空间: 不同频率分量近似独立                                    │   │
│  │                                                                     │   │
│  │   低频 ◄──── 独立 ────► 中频 ◄──── 独立 ────► 高频                 │   │
│  │                                                                     │   │
│  │   ✅ MHF 适用 (头独立假设成立)                                      │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
