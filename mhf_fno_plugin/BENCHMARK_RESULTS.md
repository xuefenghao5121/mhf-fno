## 详细分析

### 参数效率 vs 精度权衡

```
参数减少比例 vs L2 误差增加

n_heads=2:  -15.3% 参数 → +1.3% L2  ✅ 推荐精度优先
n_heads=4:  -22.9% 参数 → +4.9% L2  ✅ 推荐平衡配置
n_heads=8:  -26.7% 参数 → +6.0% L2  ⚠️ 参数效率优先
```

### 推理速度优势

- MHF-FNO 推理延迟减少 **~9%**
- 原因：更小的频域矩阵乘法

### 训练稳定性

- FNO 收敛曲线更平滑
- MHF-FNO 前期波动较大（尺度多样性初始化效果）
- 最终两者都收敛到相近精度

## 结论

### 主要发现

1. **参数效率**: MHF-FNO 成功减少参数量 15-27%
2. **精度权衡**: 参数减少带来轻微精度损失 1-6%
3. **推理速度**: MHF-FNO 推理更快约 9%
4. **最佳配置**: `n_heads=2` 提供最佳精度-效率平衡

### 推荐配置

| 场景 | 推荐配置 | 预期效果 |
|------|----------|----------|
| **精度优先** | `n_heads=2` + 尺度多样性 | L2 +1.3%, 参数 -15% |
| **平衡选择** | `n_heads=4` + 尺度多样性 | L2 +4.9%, 参数 -23% |
| **参数效率优先** | `n_heads=8` + 尺度多样性 | L2 +6.0%, 参数 -27% |

## 测试数据集说明

### Darcy Flow

- **方程**: ∇·(a(x)∇u(x)) = f(x)
- **类型**: 二阶椭圆 PDE
- **分辨率**: 16x16
- **特点**: 经典基准，适合快速验证

### Burgers Equation (未测试)

- **方程**: ∂u/∂t + u∂u/∂x = ν∂²u/∂x²
- **类型**: 一维对流扩散
- **特点**: 非线性 PDE

### Navier-Stokes (未测试)

- **方程**: ∂u/∂t + (u·∇)u = -∇p + ν∇²u
- **类型**: 二维流体动力学
- **特点**: 复杂非线性，数据集大
