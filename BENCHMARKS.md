# MHF-FNO 可用 Benchmark 数据集

## 📊 NeuralOperator 官方数据集

### 1. Darcy Flow (达西流) ✅ 已测试

**物理问题**: 多孔介质中的流体流动

**控制方程**: -∇·(a(x)∇u(x)) = f(x)

**数据集配置**:

| 参数 | 值 |
|------|-----|
| 分辨率 | 16×16, 32×32, 421×421 |
| 训练样本 | 1,000 |
| 测试样本 | 100-200 |
| 时序 | ❌ |
| 难度 | 低 |

**MHF-FNO 测试结果** (16×16):

| 模型 | 参数量 | L2误差 |
|------|--------|--------|
| FNO-32 | 133,873 | 0.0961 |
| MHF-FNO | 72,433 | 0.0919 (-4.4%) ✅ |
