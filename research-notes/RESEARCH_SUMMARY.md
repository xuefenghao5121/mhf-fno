## 理论解释

### 椭圆型 PDE 的数学特性

**1. 特征函数正交性**

```
∫ φ_n(x) φ_m(x) dx = δ_{nm}
```

→ 不同频率分量独立

**2. 特征值快速增长**

```
λ_n ~ n²  (Laplace 算子)
```

→ 高频能量被强烈抑制

**3. 频率解耦性**

```
∂²u/∂k_i∂k_j ≈ 0
```

→ MHF 头可以独立处理

### MHF 假设匹配

| MHF 假设 | 椭圆型 PDE 特性 | 实验测量 |
|----------|-----------------|----------|
| 频率独立 | 特征函数正交 | 耦合度 2.4% ✅ |
| 低频主导 | λ_n ~ n² | 能量集中在低频 ✅ |
| 弱耦合 | 扩散主导 | 平均耦合 < 0.03 ✅ |

### NS 为什么受限？

**中-高频耦合度 = 0.866** ← 这是湍流能量级联的直接证据！

```
对流项 (u·∇)u → 三波相互作用
k₁ + k₂ = k₃

能量从大尺度传递到小尺度
→ 频率全局耦合
→ MHF 独立假设失效
```
